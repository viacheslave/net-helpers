# .NET helpers and data structures

## Red Black Tree

Provides a Red-Black Tree with basic operations:
  - Insert element (key/value pair).
  - Delete element by key.
  - Count
  - IsEmpty.
  - Clear.
  - First element.
  - Last element.
  - Floor (with or without self).
  - Ceiling (with or without self).
  - Contains by key.
  - Accessor.
